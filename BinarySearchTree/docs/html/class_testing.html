Демонстрация построения Идеально Сбалансированного Дерева Поиска

Демонстрация построения ИСДП

Строит ИСДП из фиксированного набора данных и показывает:

  • Структуру дерева
  • Проверку свойств BST
  • Высоту и размер

Создает пример идеально сбалансированного дерева из отсортированных данных. Демонстрирует структуру дерева ASCII-графикой и проверяет основные свойства:

  • Корректный размер (10 узлов)
  • Высота (4 уровня)
  • Свойство BST (упорядоченность)
  • Корректность поиска
