◆ checkMemoryLeaks()

int MemoryUtils::checkMemoryLeaks ( )
static

Проверка утечек памяти

Возвращает
Количество неосвобожденных узлов

В режиме DEBUG_MEMORY:

  1. Проверяет, остались ли неосвобожденные узлы
  2. Выводит подробную информацию об утечках
  3. Возвращает количество утечек

В релизном режиме всегда возвращает 0.

Заметки
Вызывайте эту функцию перед завершением программы для проверки.

Аргументы
nodeУказатель на удаляемый узел

◆ emergencyCleanup()

void MemoryUtils::emergencyCleanup ( )
static

Аварийная очистка всей памяти

В режиме DEBUG_MEMORY:

  1. Принудительно удаляет все отслеживаемые узлы
  2. Очищает множество allocatedNodes
  3. Выводит отчет об очистке

Используется для аварийного завершения программы или в тестах. В релизном режиме не выполняет никаких действий.

Предупреждения
Эта функция должна использоваться только в крайних случаях, так как нарушает обычный жизненный цикл объектов.
