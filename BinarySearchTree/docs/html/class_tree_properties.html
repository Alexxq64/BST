Вспомогательная функция для in-order обхода

Рекурсивная реализация in-order обхода

Аргументы
nodeТекущий узел
resultВектор для сохранения результата

Рекурсивно выполняет обход в порядке: левое поддерево -> узел -> правое поддерево.

Алгоритм:

  1. Рекурсивно обойти левое поддерево
  2. Посетить текущий узел (добавить ключ в результат)
  3. Рекурсивно обойти правое поддерево

◆ searchNode()

TreeNode * TreeProperties::searchNode ( TreeNode * root,
int key )
static

Поиск узла с заданным ключом в дереве поиска

Поиск узла в дереве поиска

Аргументы
rootУказатель на корень дерева поиска
keyКлюч для поиска
Возвращает
Указатель на найденный узел или nullptr если узел не найден
Заметки
Функция работает только для деревьев поиска (BST). Временная сложность: O(h), где h - высота дерева.

Использует свойства BST для эффективного поиска:

  • Если ключ равен ключу текущего узла - узел найден
  • Если ключ меньше - ищем в левом поддереве
  • Если ключ больше - ищем в правом поддереве

Временная сложность зависит от высоты дерева: O(h)

Поиск ключа в B-дереве (ДБД).

Аргументы
rootКорень B-дерева
keyКлюч для поиска
Возвращает
Указатель на узел, содержащий ключ, или nullptr, если не найден
Заметки
Использует рекурсивный обход всех узлов B-дерева
Аргументы
rootКорень B-дерева
keyКлюч для поиска
Возвращает
Указатель на узел, содержащий ключ, или nullptr, если не найден

Выполняет поиск ключа в B-дереве с помощью рекурсивного обхода. Алгоритм:

  1. Проверяет ключи текущего узла
  2. Если ключ найден - возвращает узел
  3. Иначе рекурсивно ищет в дочерних узлах
Заметки
В B-дереве ключи внутри узла не обязательно отсортированы, поэтому используется линейный поиск по ключам узла.
