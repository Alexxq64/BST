Теоретическая оценка средней высоты ИСДП

Аргументы
nКоличество узлов в дереве
Возвращает
Теоретическая оценка средней высоты ИСДП
Заметки
Для ИСДП средняя высота близка к log₂(n) - 1
Аргументы
nКоличество узлов в дереве
Возвращает
Теоретическая средняя высота узлов

Вычисляет среднюю глубину узлов в идеально сбалансированном дереве. Алгоритм:

  1. Заполняет уровни дерева максимально возможным количеством узлов
  2. Суммирует произведение уровня на количество узлов на этом уровне
  3. Делит сумму на общее количество узлов
Заметки
Использует long double для точности при больших n

Теоретическая высота B-дерева

Высота B-дерева оценивается по формуле: h ≤ ⌈log_m(n)⌉, где m = минимальное количество ключей на узел Для t-дерева: m = ceil(t/2)
