Вывод in-order обхода дерева с репрезентативной выборкой

Аргументы
rootУказатель на корень дерева
titleЗаголовок для вывода

  • Первые 5 элементов
  • 5 элементов из середины, разделенных через ...
  • Последние 5 элементов

◆ printTreeBracket()

void OutputUtils::printTreeBracket ( TreeNode * root)
static

Вывод дерева в виде скобочной записи

Аргументы
rootУказатель на корень дерева

Формат: (корень левое-поддерево правое-поддерево) Пример: (5 (3 (1) (4)) (8 (7) (9)))

Скобочная запись удобна для визуализации структуры дерева. Формат: (корень левое-поддерево правое-поддерево) Пустые поддеревья не выводятся.
